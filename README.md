## Tech Stack

**Client:** Expo, React-Native, React, React Native Navigation, TailwindCSS, Firebase

**Database:** Firestore

## Run Locally

Clone the project

```bash
  git clone https://github.com/abhirupkumar/signal-clone.git
```

Go to the project directory

```bash
  cd signal-clone
```

```bash
  cd client
```

Install dependencies

```bash
  npm install
```
